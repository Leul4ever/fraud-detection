# Fraud Detection for E-commerce and Bank Transactions

## Overview
This project focuses on building accurate and robust fraud detection models for e-commerce and banking transactions at Adey Innovations Inc.

## Structure
- `data/`: Raw and processed datasets.
- `notebooks/`: EDA, feature engineering, and modeling notebooks.
- `src/`: Source code for the project.
- `tests/`: Unit tests.
- `models/`: Saved model artifacts.
- `scripts/`: Helper scripts.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Explore the notebooks in `notebooks/`.
