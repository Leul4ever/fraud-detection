# Notebooks
This directory contains Jupyter notebooks for each phase of the project.
- `eda-fraud-data.ipynb`: EDA for e-commerce data.
- `eda-creditcard.ipynb`: EDA for credit card data.
- `feature-engineering.ipynb`: Feature engineering steps.
- `modeling.ipynb`: Model training and evaluation.
- `shap-explainability.ipynb`: Model interpretation using SHAP.
