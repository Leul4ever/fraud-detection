# Notebooks

This directory contains Jupyter notebooks for exploration, feature engineering, and modeling.

- **eda-fraud-data.ipynb**: Exploratory Data Analysis for the E-commerce Fraud dataset.
- **eda-creditcard.ipynb**: Exploratory Data Analysis for the Bank Credit Card dataset.
- **feature-engineering.ipynb**: Geolocation mapping and feature extraction for Fraud_Data.
- **modeling.ipynb**: Model building, training, and 5-fold cross-validation.
- **shap-explainability.ipynb**: Model interpretability using SHAP (Planned).
- **data-transformation.ipynb**: Scaling, encoding, and SMOTE implementation.
